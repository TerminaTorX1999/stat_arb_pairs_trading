# Statistical Arbitrage / Pairs Trading Strategy

This repository contains a Python implementation of a mean-reversion statistical arbitrage strategy using pairs trading.

## Strategy Overview
- Cointegration-based pair selection
- Z-score driven entry and exit signals
- Dynamic hedge ratio estimation
- Basic risk controls including drawdown monitoring

## Features
- Automated signal generation
- Mean-reversion spread construction
- Simple backtesting framework
- Designed for extensibility and integration into trading systems
